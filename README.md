# 🎵 Spotify Clone 🎶

A **Spotify-inspired music player web app** built using **HTML**, **CSS**, and **JavaScript**.  
This project allows you to **play/pause songs**, and enjoy a simple **Spotify-like user interface**.

---

## 🚀 Live Demo:

👉 [Click here to view the Live Project](https://mahak-shar.github.io/spotify_clone/)

*(Hosted using GitHub Pages)*

---

## 🛠️ Tech Stack:

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## ✨ Features:

- 🎧 Play and Pause music
- ⏭️ Next and Previous song navigation
- 🕒 Seekbar / Progress bar to show current playing time
- Spotify-themed UI
